# SelfMadeAroma
SelfMadeAromaアプリは自分が作ったアロマをスマホ内に保存できるアプリです。
その他の機能としてアロマを作るうえで必要な精油の情報(柑橘系などの系統や揮発度など)をまとめたリストを閲覧することができます。
